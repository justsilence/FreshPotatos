# Fresh Potatos

## Backend

### Notes

**Please set environment first**

Create a `.env` file in `backend` folder, put your 

* `USERNAME=<REPLACE>`
* `PASSWORD=<REPLACE>`
* `HOST=<REPALCE>`
* `DATABASE=web_final`

into `.env` file.

1. make sure you enter the correct folder. (Run `cd backend`0
